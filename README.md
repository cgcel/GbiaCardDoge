# 卡好刷🐶 (GbiaCardDoge)

## 简介

个人练手项目

- Kotlin + Jetpack Compose 构建
- Material 3 UI
- 摆脱微信限制, 极速的亮码体验

## 基础功能

- [x] 手机验证码登录
- [x] 二维码显示
- [x] 二维码刷新
- [x] 深浅主题切换
